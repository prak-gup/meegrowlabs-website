What does 200 mean?

Lesson 4 · Talk to the Internet · Zero to AI Hero by Meegrow Labs

An HTTP status code is a three-digit number the server returns to describe the outcome of a request — 200 means OK, 404 means not found, and 500 means a server error occurred.
