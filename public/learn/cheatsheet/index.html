Everything a beginner needs to start commanding a computer & AI — Hindi + English.

1 Terminal essentials

pwdwhere am I?
मैं अभी कहाँ हूँ
lslist what's here
यहाँ क्या-क्या है
cd foldergo into a folder
फोल्डर के अंदर जाओ
cd ..go back up one level
एक स्तर ऊपर
cd ~go home
होम पर जाओ
mkdir namemake a folder
नया फोल्डर बनाओ
touch file.txtmake a file
नई फाइल बनाओ
cat fileread a file
फाइल पढ़ो
cp a bcopy · mv a b move/rename
कॉपी · मूव/रिनेम
rm filedelete (no undo!)
हटाओ — वापस नहीं आएगा
grep word filefind text inside a file
फाइल में शब्द ढूँढो
💻 On Windows? Install Git Bash or WSL once — then every command above works the same.

3 Talking to AI (the skill that 10x's you)

  • Be specific. "Fix my code" → "This function returns undefined when the list is empty. Make it return 0 instead."
  • Give context. Tell it the language, the goal, and what you already tried.
  • Ask it to explain, not just do — that's how you learn.
  • Iterate. First answer wrong? Tell it what's off and refine — don't start over.
  • Claude Code: install with npm i -g @anthropic-ai/claude-code, then just talk in plain English.
